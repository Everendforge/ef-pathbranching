# PathBranching MVP

The first PathBranching MVP should prove the authoring and export loop without trying to become a full production suite.

## Goals

- Create nodes.
- Create choices between nodes.
- Define simple variables.
- Attach conditions and consequences.
- Reference canon entities by stable ID.
- Export a runtime package compatible with Everend Spec v0.1.
- Validate missing canon refs and broken graph transitions.

## Non-goals

- No real-time collaboration.
- No cloud storage.
- No engine-specific runtime code.
- No replacement for WorldNotion canon editing.
